Become a Web Developer

  • Types of web developers:
  • Develops builds and maintains contents for the web
  • Front end technologies HTML, CSS and JavaScript
  • Front end represent what in front of the users when they interact with the website
  • Back end developers is not happening in front of the users but on the server in the background
  • Full stack developer is someone who works with different layers of the web development
  • Web development have a understanding about the Whole stack
  • Stack is the set of the tools, both front and back end
  • True full stack developer is a unicorn among developers

    Back-end Developer

  • start with PHP and then explore some of the other options like Node.js, .NET, or Ruby on Rails
