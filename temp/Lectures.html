    • Text Editor I used for this project, Text Editor app for Mac OS. Text editor for plain text, code editor for C, C++, Objective-C, CSS, HTML, JavaScript, JSON, PHP, XML, Ruby and Shell.   • UTF8

    • UTF stands for Unicode Transformation Format, and 8 means it uses 8-bit blocks to represent a character. What is UTF-8? Its a character encoding capable of encoding possible characters, which is defined by Unicode.
